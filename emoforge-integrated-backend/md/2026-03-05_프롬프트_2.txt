Implement admin post list query.

Requirements:

- Create AdminPostListItemDTO
- AdminPostListItemDTO는 Post 엔티티를 참고하여 작성바람.
- Add repository query method in PostRepository
- Return Page<AdminPostListItemDTO>
- Support keyword search on title
- Use Pageable
- Update AdminPostService.getPostList() to call repository

== 결과 ==
 요구사항대로 관리자 게시글 목록 쿼리를 구현했습니다.

  변경 파일:

  - AdminPostListItemDTO.java
  - PostRepository.java
  - AdminPostService.java
  - AdminPostController.java

  적용 내용:

  1. AdminPostListItemDTO 생성

  - Post 엔티티 필드 기반으로 목록용 DTO 생성:
  - id, title, viewCount, createdAt, updatedAt, memberUuid, categoryId, adminModifiedAt, adminModifiedBy

  2. PostRepository에 관리자 목록 쿼리 메서드 추가

  - Page<AdminPostListItemDTO> findAdminPostList(String keyword, Pageable pageable)
  - JPQL new ...AdminPostListItemDTO(...) 프로젝션 사용
  - 제목 키워드 검색 지원:
      - keyword가 null/빈문자면 전체 조회
      - 아니면 title LIKE %keyword% (대소문자 무시)

  3. AdminPostService.getPostList() 수정

  - 반환 타입: Page<AdminPostListItemDTO>
  - PostRepository.findAdminPostList(keyword, requestDTO.toPageable()) 호출하도록 변경
  - 신규 쿼리 로직 없이 Repository 호출 방식으로 처리

  4. 컨트롤러 최소 연동

  - GET /api/admin/posts에 keyword 요청 파라미터를 받아 서비스에 전달하도록 수정

  검증:

  - .\gradlew :module-post:compileJava 빌드 성공했습니다.
