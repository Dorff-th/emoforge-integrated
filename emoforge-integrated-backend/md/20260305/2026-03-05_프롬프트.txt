현재 프로젝트에서 다음 리팩토링 작업을 수행하려고 한다.

작업 목록:
1. domain module들에서 spring-boot-starter-security 중복 제거
2. org.springdoc:springdoc-openapi-starter-webmvc-ui 의존성을 emoforge-core → emoforge-app으로 이동
3. jakarta.servlet-api 의존성 제거

요청:
1. 이 변경이 영향을 줄 수 있는 모든 파일을 찾아라.
2. 변경이 필요한 build.gradle 파일 목록을 제시하라.
3. Security 관련 import 또는 bean 설정에 영향이 있는 Java 파일이 있는지 확인하라.
4. 각 변경에 대한 예상 리스크를 설명하라.

코드를 수정하지 말고 분석 결과만 보고하라.
