관리자가 게시글을 수정을 했다면 

관리자가 2026-03-05 14:32:11 수정 이렇게 흔적남기기를 위한 post 테이블 필드 추가

ALTER TABLE post
ADD COLUMN admin_modified_at DATETIME NULL COMMENT '관리자 수정 시각' AFTER updated_at,
ADD COLUMN admin_modified_by VARCHAR(50) NULL COMMENT '수정 관리자' AFTER admin_modified_at;

Admin 수정 로직 예시 (참고)

UPDATE post
SET
    title = ?,
    content = ?,
    admin_modified_at = NOW(),
    admin_modified_by = 'admin'
WHERE id = ?
