
# Emoforge Frontend

React + TypeScript 기반 SPA 애플리케이션입니다.

---

## ⚙ 기술 스택

- React
- TypeScript
- Vite
- TailwindCSS
- React Router
- Zustand
- React Query

---

## 📂 주요 구조

```text
src/
├─ features/
├─ pages/
├─ components/
├─ hooks/
├─ api/
└─ router/
```


---

## 🚀 실행

```bash
npm install
npm run dev
```

빌드:
```bash
npm run build
🌍 환경 변수
```
.env 파일에:

VITE_API_BASE_URL=
VITE_KAKAO_REDIRECT_URI=
