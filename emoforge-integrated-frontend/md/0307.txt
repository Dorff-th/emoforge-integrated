관리자 게시판 frontend 구현을 위한 사전작업

백엔드 API 엔드포인트
GET	/api/posts/admin/posts	 	관리자 페이지에서 게시판 목록 조회
GET	/api/posts/admin/posts/{id} 	관리자 페이지에서 게시물 생세 조회
PUT  /api/posts/admin/posts/{id} 	관리자 페이지에서 게시물 수정 처리
DELETE	/api/posts/admin/posts/{id} 		관리자 페이지에서 단일 게시물 삭제 처리
DELETE	/api/posts/admin/posts	/bulk		관리자 페이지에서 게시물 일괄 삭제 처리

GET		/api/posts/admin/{postId}/comments		관리자 페이지에서 특정 게시물의 댓글 목록 조회
DELETE	/api/posts/admin/{postId}/comments/{commentId}  관리자 페이지에서 특정 게시물의 댓글 삭제 처리
DELETE	/api/posts/admin/{postId}/comments/bulk	관리자 페이지에서 특정 게시물의 댓글 일괄 삭제 처리
